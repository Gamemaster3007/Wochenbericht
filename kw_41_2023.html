Was ist html?

HTML steht für "Hypertext Markup Language" und ist die Standardauszeichnungssprache für Webseiten. Es handelt sich um eine Schlüsseltechnologie des World Wide Web und bildet das Grundgerüst einer Webseite, indem sie deren Struktur und den Textinhalt definiert. HTML verwendet spezielle Tags (Auszeichnungen), um Elemente auf einer Webseite zu beschreiben, wie Überschriften, Absätze, Bilder, Links und viele andere.

Die Bildlegende deklariert/beschreibt den Bildinhalt und enthält die Bildquelle und Urheberschaft.

Was ist css?

CSS steht für "Cascading Style Sheets" und ist eine Stylesheet-Sprache, die in Kombination mit HTML verwendet wird, um das Aussehen und Layout von Webseiten zu gestalten. CSS ermöglicht es Webdesignern und Entwicklern, das visuelle Erscheinungsbild von Webseiten festzulegen, einschließlich Farben, Schriftarten, Abstände, Positionen von Elementen und viele andere stilistische Aspekte. Im Wesentlichen trennt CSS die Präsentation (Styling) von HTML (Struktur und Inhalt), was eine klare Trennung der Aufgaben in der Webentwicklung ermöglicht.

Was habe ich alles mit css gemacht?

Diese Woche durften wir auch zum Teil mit css arbeiten. Vor allem, wenn es um das Gestalten unser eigener Website ging.
